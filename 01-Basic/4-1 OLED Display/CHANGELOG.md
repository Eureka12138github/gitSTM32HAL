## 🎯 使用建议

### 推荐用法

```bash
# 在任何 PIO 项目中
双击 sync.bat

# 脚本会自动：
1. 检测当前项目目录
2. 验证项目结构
3. 同步到正确的目录
```

### 注意事项

- ⚠️ 脚本必须放在 PIO 项目根目录
- ⚠️ 项目必须包含 `src/` 和 `platformio.ini`
- ⚠️ CubeMX 路径需在脚本中配置

## 📞 技术支持

### 遇到问题？

1. **检查路径检测**

   ```bash
   # 应该看到：
   [AUTO-DETECT] Using current directory: ...
   ```

2. **验证项目结构**

   ```bash
   # 必须有：
   - src/
   - platformio.ini
   ```

3. **查看文档**
   - `AUTO_DETECT_README.md` - 详细说明
